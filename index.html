How to Install a PHP Web Based Appication

Now don't worry, at the beggining of the semester I didn't know what a PHP was either. A PHP is a CMS system like Joomla and WordPress. Today I will be explain how to install WordPress through Amazon Lightsail.

Create a Lightsail account

  • The first step to this process is to go to lightsail.awa.amazon.com and create an account for yourself.
  • Once you create your account you are going to want to create an instance

Creating an instance

This is where you start to build your instance and hopefully get it up and running

  • Click the create an instance button
  • When brought to the screen with the instances: click and make sure your data region is correct
  • Next you will want to select the platform Linus/Unix
  • Then you will choose the WordPress instance. Choosing the WordPress blueprint will set up WordPress for you and you won't have to go through the steps of setting it up manually.
  • Now, you do hae to pay for lightsail. It's amazon, nothing is free! Choose the cheatest option which is $3.50 a month with the first month free.
  • The last part of the instance building is identifying your instance. You want to make sure you choose a unique name for your instance but make it something recognizable.
  • Click create instance!

Accessing WordPress

Now, here is where it can get slightly confusing. Stay with me now.

  • Click on your WordPress Instance
  • To find what your username and password is, you will want to click connect using SSH for your password, and your username will be below (called Bitnami) along with your public IP.
  • Once you open the large black screen that says bitnami with all the code on the page, you will want to type in this exact code to retrieve your password
